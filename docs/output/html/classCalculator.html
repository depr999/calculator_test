Detailed Description

Основной класс калькулятора с графическим интерфейсом

Основной класс калькулятора

Класс реализует полнофункциональный калькулятор с поддержкой базовых арифметических операций: сложение, вычитание, умножение и деление. Поддерживает ввод многозначных чисел и последовательные вычисления.
